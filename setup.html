Setup

Setup

Overall

  • Set up a jsconfig.json or tsconfig.json that include all vue files and files that they import from, for example:

  • jsconfig.json

      {
    "include": [
      "./src/**/*"
    ]
  }

  • tsconfig.json

      {
    "include": [
      "./src/**/*"
    ],
    "compilerOptions": {
      "module": "es2015",
      "moduleResolution": "node",
      "target": "es5",
      "sourceMap": true,
      "allowJs": true
    }
  }

ESLint

  • Install ESLint extension

  • Add vue to your eslint.validate setting, for example:

    "eslint.validate": [
  "javascript",
  "javascriptreact",
  "vue"
]

Sass

  • Install Sass extension for Sass support

Stylus

  • Install Stylus extension for Stylus support
